# 🚀 Проект на Next.js (Full‑Stack)  
*Учебный проект по созданию полноценного веб-сайта с использованием Next.js*

---

## 📖 О проекте  
Этот проект разрабатывается в рамках учебного курса.  
Главная цель – **освоить full‑stack разработку на базе Next.js (App Router) и Express**.

**Тематика:** рассказать о профессии инженера по проектированию сетей, о ключевых нужный навыков, что он должен делать
**Стиль и визуал:** темная тема, текст белый, ключевые слова выдели градиентом сине-фиолетовым, дизайн на пк сетка, на мобильном в виде стобца
**Целевая аудитория:** студенты, который ищут себя и свою профессию
